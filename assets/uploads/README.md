# Bilder-Ordner (Uploads)

In diesem Ordner landen automatisch alle Bilder, die über das CMS
(`/admin/`) hochgeladen werden.

- Du kannst Bilder auch direkt hier ablegen.
- Die Beschreibungen/Bildunterschriften werden im CMS gepflegt
  (Galerie, Team, Ressorts) und in den Dateien unter `content/` gespeichert.
- Diese README sorgt nur dafür, dass der Ordner in Git erhalten bleibt.
